## 📌 Project Overview

This project is a modern, responsive frontend built as part of the **Praava Legal task**.

It focuses on:
- Clean UI/UX ✨  
- Responsive design 📱  
- Smooth animations ⚡  
- Dark mode support 🌙  

Built using **Next.js** with a focus on performance and scalability.

---

## 🛠️ Tech Stack

- ⚛️ **Next.js** (App Router)
- 🎨 **Tailwind CSS**
- 💡 **TypeScript**
- 🌗 **Dark Mode UI**
- 🚀 **GitHub Pages Deployment (CI/CD)**

---

## ✨ Features

- 🌟 Modern floating hero section
- 🌙 Dark mode support
- 📱 Fully responsive design
- ⚡ Smooth animations & transitions
- 🧩 Clean component structure
- 🚀 Deployed using GitHub Actions
